# HA Fleet Manager Integration

Monitor and manage your Home Assistant instance from a centralized Fleet Manager dashboard.

## Quick Start

1. **Install this integration** via HACS
2. **Add to configuration.yaml:**
   ```yaml
   ha_fleet:
     api_url: "http://YOUR_BACKEND:8100"
     api_token: "YOUR_TOKEN"
   ```
3. **Restart Home Assistant**
4. **View in dashboard** - Your instance appears automatically!
